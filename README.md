# chunker

Personal learning project on chunked file uploads — slicing a file into chunks in the browser, uploading them, and reassembling the file on the server.

## Tech stack

- Next.js 16.2.10 (App Router)
- React 19.2.4
- TypeScript
- Tailwind CSS v4

## Getting started

```bash
npm install
npm run dev
```

App runs at http://localhost:3000.

## Scripts

- `npm run dev` — dev server
- `npm run build` — production build
- `npm run start` — run the production build
- `npm run lint` — ESLint
